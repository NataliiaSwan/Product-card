Natural Self-Care Products - HTML/CSS/JS Template

Опис:
Це готовий шаблон сторінки продажу продуктів для self-care. 
Має адаптивні картки, модальні вікна з формою для замовлення, кнопки Buy Now і Learn More.

Функціонал:
- Адаптивний дизайн (desktop, tablet, mobile)
- Модалки для замовлення з формою (ім'я, email)
- Кнопки Buy Now і Learn More
- Картки з назвою, описом, ціною, зображенням та фічами
- Всі картинки мають alt-теги для SEO
- Готовий до швидкого старту продажу або демонстрації

Файли в пакеті:
- index.html       ← головна сторінка
- src/style.css     ← CSS стилі
- src/index.js      ← JS скрипт для модалок і кнопок
- images/           ← локальні копії зображень (за потреби)
- README.txt        ← ця інструкція

Інструкція по використанню:
1. Розпакуйте архів.
2. Відкрийте index.html у браузері або завантажте на хостинг (Netlify, Vercel, GitHub Pages).
3. Щоб змінити картинки, замініть їх у папці images/ і оновіть посилання у index.html та index.js.
4. Для форми замовлення можна підключити Formspree або EmailJS для отримання email.
5. Стилі можна змінювати у style.css, JS - у index.js.
6. Для live demo додайте посилання на свій хостинг (наприклад: https://natural-self-care-products.netlify.app/)

Ліцензія:
- Можна використовувати для комерційних та особистих проектів.
- Заборонено перепродавати цей шаблон як свій без змін.
